## 변수 (Variable)

- 값을 저장할 수 있는 메모리의 특정 번지에 붙여진 이름
- 변수를 통해 메모리 번지에 값을 저장할 수 있음
- 변수는 정수, 실수 등 다양한 타입의 값을 지정할 수 있음

## 리터럴 (literal)

- 소스 코드에서 직접 입력된 값

## 상수

- 상수는 변경이 불가능함

- final를 추가하면 상수 생성 가능

- 상수의 명칭은 대문자로만 작성

```java
final int NUM;
```
